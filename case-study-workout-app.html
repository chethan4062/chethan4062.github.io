WORKOUT SEQUENCER
Personal project · 2026
CASE STUDY
A workout app built to be looked at as little as possible.
A weekly lifting + Zone 2 plan, sequenced into one tap-to-advance flow — so the phone stays out of the way and the plan stays in view.
Design Component
Mobile-first
Local persistence, no backend
01 — THE BRIEF
A fixed 7-day plan already existed — lower/upper splits plus Zone 2 cardio. The app's only job was to sequence it.
No logging screens, no rep-by-rep data entry. Used mid-set with sweaty hands, so the core loop had to be one deliberate tap: mark the set done, and the app tells you what's next — exercise, reps, RPE, rest.
02 — WHAT WE BUILT
SEQUENCED SESSIONS
What follows what, always visible
Every session shows the current exercise's sets, reps and RPE up top, with the full remaining order queued below as "Up next" — never just one step ahead. A progress dial across the top tracks position in the session at a glance.
Finishing a set auto-starts a rest countdown sized to that exercise, then hands the same screen back — no navigating, no re-finding your place.
GENERATED FORM GUIDE
A movement diagram, with no photoshoot required
Rather than leave a blank spot for a demo photo, each exercise name is matched to one of nine movement families — squat, hinge, horizontal press, overhead press, pull, curl, lateral raise, tricep extension, calf raise, core brace — and rendered as a small looping stick-figure animation, generated live from CSS.
It's schematic on purpose: a directional cue for the movement pattern, not a replacement for real coaching.
ZONE 2, NOT ZONE 5
A different mode for a different kind of day
Cardio days swap the set-by-set flow for one large countdown and an adjustable target heart-rate band — nothing to tap until the timer runs out.
A HISTORY THAT DOESN'T LIE
Calendar log, snapshotted per day
Every completed day is stamped green on a real month calendar. Tapping a date shows exactly what ran that day — a frozen snapshot of the routine at completion time, so editing a routine later never rewrites history.
WEEK-AWARE PLANNING
Plan the week ahead, lock the week behind
The Plan screen reassigns which routine runs on which day — for a specific, dated week, not a floating template. Past weeks lock outright; within the live week, any day already lifted or already gone is locked too. A library of routines can be edited or extended with a simple set builder.
03 — PRINCIPLES
One tap, always forward
Mark set done is the only required action in a lift session.
Zero manual logging
No weight fields, no rep counters to fill in — completion is inferred from progress through the plan.
Stark, high-contrast type
Inter for copy, JetBrains Mono for numbers — legible at arm's length, mid-set.
History can't be rewritten
Snapshots, locks, and past-week guards keep the record honest.
Built as a single streaming Design Component — no backend, progress kept on-device.
